**Datum:** 2025-12-03
**Entwickler:** Claude (AI Assistant)
**Sprint:** P1 - Core Frontend

## Problem

Auf der Kaspa-Seite war der Text "KASPA" + Tagline AUF dem Banner-Bild als Overlay positioniert. Dies war inkonsistent mit dem Design-Konzept.

**Vorher:**
```
┌─────────────────────────────────────┐
│  BANNER (Bild)                      │
│  ┌──────────────────┐               │
│  │ [K] KASPA        │               │
│  │ Die schnellste.. │               │
│  └──────────────────┘               │
└─────────────────────────────────────┘
```

---

## Loesung

Text aus dem Banner entfernt und in eine separate GlassCard unter dem Banner verschoben.

**Nachher:**
```
┌─────────────────────────────────────┐
│  BANNER (nur Bild)                  │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│ [K] KASPA                           │
│ Die schnellste Proof-of-Work...     │
└─────────────────────────────────────┘
```

---

## Technische Aenderungen

### Banner (nur Bild)
- Text-Overlay komplett entfernt
- Gradient geaendert: `bg-gradient-to-r` zu `bg-gradient-to-t`
- Gradient-Opacity reduziert: von `/70` auf `/30`
- Cleaner Look ohne Textüberlagerung

### Token Header (neue GlassCard)
- Separates GlassCard-Element unter dem Banner
- Flex-Layout mit Icon und Text
- Icon: 56px/64px (mobile/desktop) mit Kaspa-Gradient
- Titel: KASPA (bold, 2xl/4xl)
- Tagline: i18n Text `{t('tagline')}`
- Responsive Padding: p-4 (mobile) / p-6 (desktop)
- margin-bottom: 6 (mb-6)
