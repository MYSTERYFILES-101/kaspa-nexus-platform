# P1.X-FIX PROTOKOLL - Feste Container-Groessen

```
+==============================================================+
|  PROTOKOLL: P1.X-FIX - Container Max-Width                   |
|  Datum: 2025-12-02                                           |
|  Status: ABGESCHLOSSEN                                       |
+==============================================================+
```

## PROBLEM

Boxen und Container skalierten unbegrenzt mit der Bildschirmbreite.
Auf grossen Monitoren wurden die Inhalte zu breit und schwer lesbar.

## LOESUNG

Feste max-width fuer Content-Container:
- `--container-max-width: 1400px`
- `--content-max-width: 1200px`
- `margin: 0 auto` (zentriert)

## DURCHGEFUEHRTE AENDERUNGEN

### 1. CSS Variables (globals.css)
```css
/* Container */
--container-max-width: 1400px;
--content-max-width: 1200px;
```

### 2. Content Container Class (globals.css)
```css
.content-container {
  max-width: var(--container-max-width);
  margin: 0 auto;
  padding: 0 1rem;
}

@media (min-width: 768px) {
  .content-container { padding: 0 1.5rem; }
}

@media (min-width: 1024px) {
  .content-container { padding: 0 2rem; }
}
```

### 3. AppLayout.tsx
Main Content umschliesst jetzt children mit content-container:
```tsx
<main className="main-content">
  <div className="content-container">
    {children}
  </div>
</main>
```

### 4. Pages angepasst
Padding von `p-4 md:p-6 lg:p-8` zu `py-4 md:py-6 lg:py-8`
(nur vertikales Padding, horizontales kommt vom Container)

- `src/app/page.tsx`
- `src/app/krc20/tokens/page.tsx`
