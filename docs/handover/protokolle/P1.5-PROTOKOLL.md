## Komponenten

### 1. DEX Page (`page.tsx`)

**Header:**
- 21:9 Banner (Banner-Investment-Hub.png als Platzhalter)
- GlassCard mit Titel und Subtitle
- DEX Icon (Swap-Symbol)

**Statistik-Karten:**
- Total DEXes
- Active DEXes
- Trading Pairs
- 24h Volume (Coming Soon)

**DEX Liste:**
3 DEXes mit Kacheln:
1. **KSPR Bot** - Telegram Bot fuer KRC-20 Swaps
2. **Chainge Finance** - Cross-Chain DEX mit KAS Bridge
3. **KasPlex** - Token Deploy, Mint, Transfer

**Jede DEX-Kachel zeigt:**
- Logo/Placeholder
- Name
- Type Badge (Telegram Bot / Web App)
- Status Badge (Active/Maintenance/Offline)
- 24h Volume (placeholder)
- Anzahl Pairs
- Feature Tags
- Visit Link

**Info Sektionen:**
1. DEX vs CEX Vergleich
   - DEX Vorteile (4 Punkte)
   - CEX Bedenken (4 Punkte)
   - Info-Box mit Empfehlung

2. How to Trade
   - 3-Schritte Anleitung
   - Wallet verbinden, Token auswaehlen, Swap bestaetigen

### 2. Loading State (`loading.tsx`)

Skeleton-Animationen fuer:
- Banner
- Header
- Stats Grid
- DEX Cards (3x)
- Info Sections
- How to Trade Section

## Design-Entscheidungen

1. **Placeholder Logo**: Gradient-Kreis mit erstem Buchstaben statt Bild
2. **Status Badges**: Farbcodiert (green=active, yellow=maintenance, red=offline)
3. **Type Badges**: Icon + Text fuer Telegram/WebApp
4. **Hover Effects**: Glow + Border-Farbaenderung
5. **Responsive**: 1 Spalte mobil, 2-3 Spalten Desktop
