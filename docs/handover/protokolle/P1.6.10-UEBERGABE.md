## Aufgabe

Integration des offiziellen Kaspa SVG-Logos in die KRC-20 Switchbar anstelle des generischen Icons.

## Durchgefuehrte Aenderungen

### 1. Logo kopiert

**Quelle:** `/root/Logo&Banner/kaspa-kas-logo.svg`
**Ziel:** `/home/kaspa/kaspa-nexus-platform/public/images/logos/kaspa-logo.svg`

Das Logo ist das offizielle Kaspa-Logo:
- Tuerkiser Kreis (#6fc7ba)
- Schwarzes "K"-Symbol (#1a1a1a)
- SVG-Format (skalierbar, klein)

### 2. Switchbar aktualisiert

**Datei:** `src/components/krc20/Krc20Switchbar.tsx`

Aenderungen:
- `Image` Import von `next/image` hinzugefuegt
- `KaspaIcon` Komponente ersetzt: SVG-Icon → Image mit offiziellem Logo

**Vorher:**
```tsx
const KaspaIcon = () => (
  <svg className="w-4 h-4" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path ... />
  </svg>
);
```

**Nachher:**
```tsx
const KaspaIcon = () => (
  <Image
    src="/images/logos/kaspa-logo.svg"
    alt="Kaspa"
    width={18}
    height={18}
    className="w-[18px] h-[18px]"
  />
);
```
