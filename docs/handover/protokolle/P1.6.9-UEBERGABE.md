## Aufgabe

Implementierung einer horizontalen Tab-Navigation (Switchbar) fuer den KRC-20 Bereich, die auf allen 5 Seiten angezeigt wird und schnelles Wechseln zwischen den Bereichen ermoeglicht.

**Vorlage:** Screenshot `switchbar.jpg` mit Design-Referenz

## Durchgefuehrte Aenderungen

### 1. Neue Komponente erstellt

**Datei:** `src/components/krc20/Krc20Switchbar.tsx`

Features:
- 5 Tabs: Kaspa, Tokens, DEX, DeFi, Infrastructure
- Icons fuer jeden Tab (SVG)
- Aktiver Tab hervorgehoben (cyan/tuerkis Hintergrund mit Schatten)
- Responsive: Icons auf Mobile, Icons + Text auf Desktop
- Glasmorphism-Design (glass-card)
- Automatische Erkennung der aktiven Seite via `usePathname()`
- Tokens-Tab matcht auch Token-Detailseiten (`/krc20/tokens/[ticker]`)

## Technische Details

### Komponenten-Struktur

```tsx
<Krc20Switchbar />
  └── glass-card Container (rounded-2xl, p-2)
      └── Flex Container (gap-1 md:gap-2, centered)
          └── Link Tabs (5x)
              ├── Icon (SVG, w-4 h-4)
              └── Label (hidden sm:inline)
```

### Aktiver Tab Styling

```css
/* Aktiv */
bg-[var(--color-primary)]
text-white
shadow-lg
shadow-[var(--color-primary)]/30

/* Inaktiv */
text-[var(--color-text-secondary)]
hover:text-[var(--color-text-primary)]
hover:bg-[var(--glass-bg)]
```

### Responsive Verhalten

| Breakpoint | Verhalten |
|------------|-----------|
| < 640px (sm) | Nur Icons, gap-1 |
| >= 640px | Icons + Text, gap-2 |
