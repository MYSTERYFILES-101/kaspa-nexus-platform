## AUFTRAG

Implementierung der KRC-20 Token Liste mit:
- KRC-20 Network Banner
- Token Grid mit Suche und Sortierung
- Pagination
- Live-Daten Badge
- API Integration

## DURCHGEFUEHRTE ARBEITEN

### 1. Banner Integration
- Kopiert: `/root/Logo&Banner/Krc20 netzwerk.png` -> `public/images/banners/krc20-network.png`
- 21:9 Aspect Ratio mit Gradient Overlay

### 2. Neue Komponenten erstellt

**`src/components/tokens/TokenSearch.tsx`**
- Suchfeld mit Debounce (300ms)
- Clear-Button
- Glasmorphism Design

**`src/components/tokens/TokenSort.tsx`**
- Dropdown mit 7 Sortieroptionen:
  - Market Cap (hoch/niedrig)
  - Volume (hoch)
  - Change (beste/schlechteste)
  - Holders (meiste)
  - Name (A-Z)

**`src/components/tokens/TokenPagination.tsx`**
- Prev/Next Buttons
- Page Numbers mit Ellipsis
- Max 5 sichtbare Seiten

**`src/components/tokens/TokenGrid.tsx`**
- Responsive Grid (1-4 Spalten)
- Loading Skeleton
- Empty State

**`src/components/tokens/index.ts`**
- Export-Index fuer alle Token-Komponenten

### 3. Token-Seite erstellt

**`src/app/krc20/tokens/page.tsx`**
- Route: `/krc20/tokens`
- Features:
  - Banner mit KRC-20 Network Grafik
  - Live-Daten Badge mit Update-Zeit
  - Suchfeld (Ticker/Name)
  - Sortierung (7 Optionen)
  - Pagination (24 Tokens pro Seite)
  - Error State mit Retry
  - Auto-Refresh (30s)

### 4. Translations erweitert

**`messages/de.json`** und **`messages/en.json`**
- Neuer Abschnitt `tokens` mit:
  - title, subtitle
  - liveData, lastUpdated
  - Sortieroptionen
  - Pagination Labels
  - Error/Empty States
