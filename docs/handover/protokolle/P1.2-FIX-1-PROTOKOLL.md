## PROBLEM

Seiten-Hintergrund hatte kein Netzwerk/Blockchain-Muster wie auf
der alten kaspa-nexus.io Webseite.

## ANALYSE

- Screenshot von alter Webseite analysiert (Screenshot 2025-12-02 235048.jpg)
- Subtiles hexagonales Netzwerk-Muster im unteren Bereich sichtbar
- Muster soll ueber dem Gradient-Hintergrund liegen

## LOESUNG

1. Screenshot als Hintergrund-Bild verwendet
2. Als `::before` Pseudo-Element implementiert
3. Opacity 0.4 fuer subtilen Effekt (etwas intensiver als Original)
4. Fixed positioning fuer Parallax-Effekt

## DURCHGEFUEHRTE AENDERUNGEN

### 1. Hintergrund-Bild kopiert
```
/root/Logo&Banner/Screenshot 2025-12-02 235048.jpg
-> public/images/backgrounds/network-pattern.jpg
```

### 2. CSS angepasst (globals.css)
```css
body {
  position: relative;
}

/* Network Pattern Background */
body::before {
  content: '';
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background-image: url('/images/backgrounds/network-pattern.jpg');
  background-size: cover;
  background-position: center;
  background-repeat: no-repeat;
  background-attachment: fixed;
  opacity: 0.4;
  pointer-events: none;
  z-index: 0;
}
```
